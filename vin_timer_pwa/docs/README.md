# VIN Timer PWA

Gotowa instalowalna aplikacja internetowa na telefon. Działa offline po pierwszym otwarciu.

## Uruchomienie
Pliki należy umieścić na hostingu HTTPS (np. firmowy serwer, GitHub Pages, Netlify). Otwórz adres w Chrome na Androidzie i wybierz „Dodaj do ekranu głównego” / „Zainstaluj aplikację”.

## Funkcje
- VIN 17 znaków, walidacja i blokada duplikatów
- timer 40 minut, ostrzeżenie 5 minut i alarm po czasie
- do 50 aktywnych pojazdów
- lokalizacja, operator i notatka
- przedłużanie czasu, zakończenie i usuwanie
- historia i eksport CSV
- lokalne przechowywanie, działanie offline
- skaner kodu kreskowego/QR w obsługiwanych przeglądarkach

## Ważne
Powiadomienia przeglądarkowe muszą zostać zaakceptowane przez użytkownika. System mobilny może ograniczać powiadomienia po całkowitym zamknięciu przeglądarki. Dla gwarantowanych alarmów w tle najlepsza jest natywna aplikacja Android APK.
